Try every vowel — which ones make real words?

🎮What: A word-discovery game — try every vowel in a consonant frame to find all the real words it makes.
👆How to play: You see a frame like B_T. Tap A, E, I, O, U in turn — green chips appear for real words; invalid combos flash red.
🧠Helps with: Phoneme substitution — swapping a single sound to create a new word deepens the understanding that one letter change = one meaning change.
